# 本地股票投资记录工具

用 Streamlit + SQLite 构建的本地小工具，记录交易、标的计划、板块分布，以及未来 3-6 个月的宏观/个股大事。

## 快速启动
1) 建议使用 Python 3.10+。在仓库根目录执行：
```bash
python -m venv .venv
.\.venv\Scripts\activate       # Windows PowerShell
pip install -r requirements.txt
python -m streamlit run app.py
```
2) 浏览器会打开 `http://localhost:8501`。所有数据保存在 `data/portfolio.db`，默认会创建。

## 功能概览
- **看板**：成本、市值、已/未实现盈亏，持仓明细，板块分布；支持手动更新现价。
- **交易录入**：买入/卖出、数量、价格、手续费、板块和备注。
- **标的/板块**：维护目标价、止损价、关键买卖点、投资逻辑/催化剂。
- **大事日历**：查看未来 3-6 个月的大事（宏观+个股），可自行添加。首次启动会预填示例事件。

## 数据说明
- 交易以逐笔方式存储，采用加权平均成本法计算持仓与盈亏；卖出自动计入已实现盈亏。
- 现价需手动输入（在“看板”页），可按需多次更新。
- 若需要外部行情或事件自动同步，可在此基础上接入你的行情 API/日历数据源。

## 目录
- `app.py`：主应用。
- `requirements.txt`：依赖列表。
- `data/portfolio.db`：运行时生成的本地数据库。
